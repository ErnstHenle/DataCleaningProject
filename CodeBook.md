# Code book

## Data Cleaning Project
run_analytics.R

### 0. Download data and assemble training and testing datasets
* Download "https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip"
* Combine (cbind) subject variable from subject_train.txt with training dataset
* Combine (cbind) activity variable from y_train.txt with training dataset
* Combine (cbind) subject variable from subject_test.txt with testing dataset
* Combine (cbind) activity variable from y_test.txt with testing dataset

### 1. Merges the training and the test sets to create one data set.
* Creates dataset CombinedData
* Combine (rbind) training and testing datasets
* New combined data set is CombinedData

### 2. Extracts only the measurements on the mean and standard deviation for each measurement.
* Modifies dataset CombinedData
* use grep on features from "UCI HAR Dataset/features.txt" to indentify mean and standard deviations
* Retain only those columns with mean and standard deviation

### 3. Uses descriptive activity names to name the activities in the data set
* Modifies dataset CombinedData
* substitute activity codes with activity labels from "activity_labels.txt"

### 4. Appropriately labels the data set with descriptive variable names.
* Modifies dataset CombinedData
* Use labels from "UCI HAR Dataset/features.txt"
* remove "-", and "()" from variable names and introduce Camel case

### 5. From the data set in step 4, creates a second, independent tidy data set with the average of each variable for each activity and each subject.
* Creates dataset MeanByActivitySubject
* load dplyr package
* use group_by() and summarise_each() to determine the mean for each subject and activity of each variable
* The reslut is a new tidy data set called MeanByActivitySubject

### Persist both tidy data sets as csv files in working directory
* Creates dataset files
* save the first tidy data to the current directory as CombinedData.csv
* save the second tidy data to the current directory as MeanByActivitySubject.csv
